This is a demo for an experiment on Human-AI teaming and preferences for controllability (McDonald & Gonzalez, 2025) in a reproduction of the Overcooked-AI environment (Carroll et al., 2019) that uses CoGrid and Interactive Gym.

CoGrid and Interactive Gym were developed in the Dynamic Decision Making Lab at Carnegie Mellon University. They comprise a framework for developing Human-AI interaction experiments, where CoGrid is an easy-to-use library for building grid-based environments (in the Gymnasium or PettingZoo formats) for multi-agent simulation and AI training. Interactive Gym is a library for developing and running these experiments in a web browser, it provides the novel functionality to run these Python-based environments directly in the browser.

Here's how the demo works:

  1. On the next page, you'll see the instructions for how to play the Overcooked game.
  2. The following page will allow you to choose between three different partner types and play a round of the game with them. This repeats three times.

Our goal in running the experiment corresponding to this demo is to understand how people interact with controllable AI partners in a collaborative task. We're interested in how such AI partners can be used to complement both subjective and objective preferences in human partners.

We develop and employ Behavior Shaping, an algorithm for training controllable policies in reinforcement learning.
